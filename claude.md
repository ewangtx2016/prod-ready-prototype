# claude 项目配置

## 上下文控制
- 仅读取 `src/` 和 `tests/` 目录下的文件
- 禁止自动读取 `node_modules/` 和 `public/`

## 回复约束
- 优先使用现有依赖，不提议新增 npm 包
- 修改代码前说明变更范围，确认后再执行
- 遇到不确定或者不合理或者有分歧的问题，请不要瞎编，抛除问题并给出方案和建议选择。

### 每次新的上下文窗口回复前，先阅读以下知识库文件获取上下文：
- @.claude/knowledge/project.md
- 如果修改了架构或新增规范，同步更新对应知识库文件。

## 工具限制
- 使用 Edit 工具而非全量重写文件
- 禁止运行 `npm install` 除非显式授权
